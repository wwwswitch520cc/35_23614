<p>《我的朋友小猪佩奇 My friend Peppa Pig》是Petoons Studio制作的休闲游戏，</p>
<p>以《小猪佩奇》的世界观为基础，与佩奇一同踏上有趣的冒险之旅。</p>
<p>玩家能够在自定义装扮，打造个性化的角色，认识新的朋友，探索海滩、森林等多个地点，</p>
<p>无论是小朋友还是带孩子的家长都可以享受游戏的乐趣。</p>

<p>系统要求:14.1.2</p>
